I’ve been inspired by other R packages that have created color palettes based on living organisms. Naturally I decided to try to make a color palette using prairie species. I didn’t do anything fancy when chosing colors. I merely used the instant eyedropped to hover over different pixels in the photo and wrote down the hex codes. Obviously colors will vary based on lighting and shading. All the colors and the photos can be found in the photo gallery, if a photo is taken from somewhere else then I’ve noted it as such.

For all the botany nerds out there I’ve decided to grab colors from two species in 4 different families. There are obviously WAY too many beautiful flowers to choose from. This is merely a starting point.

i. The bean family (Fabaceae)

I started with Purple Prairie Clover (Dalea purpurea), which was an obvious choice for me since it has a lovely contrasting purple/orange color profile.

dalpu_colors <- c("#5F1343", "#A41393", "#C669D5",  "#E98018", "#667056",
                  "#2E4D24")
dalpu_raw <- readJPEG("dalpu.JPG")
dalpu     <- rasterGrob(dalpu_raw, interpolate = TRUE)

dalpu_gg <- make_col_plot()

grid.arrange(dalpu, dalpu_gg, ncol = 2)

I moved onto Leadplant (Amorpha canescens) which has a similar color palette to purple prairie clover. Predominantly purple coloring with orange anthers. However, leadplant also has beautiful red styles and the overall color of the plant is a lovely gray-green. Both purple prairie clover and leadplant are in the bean family (Fabaceae).

amoca_colors <- c("#361554", "#7A4FB6", "#C7B1F2", "#E99725", 
                  "#8B1C38", "#778A77")
image(1:6, 1, as.matrix(1:6), col = amoca_colors,
      ylab = "", xlab = "", xaxt = "n", yaxt = "n", bty = "n")

ii. The milkweed family (Asclepiadaceae)

Next I chose buttefly milkweed (Asclepias tuberosa) which is one of the few orange prairie flowers. Being a milkweed species it prized for the sustenance it provides Monach butterflies larvae.

asctu_colors <- c("#F64850", "#AA230F", "#FD5D0E", "#F9AB1A", "#EFBB56", 
                   "#ECE982", "#B9E437", "#5F942B")
image(1:8, 1, as.matrix(1:8), col = asctu_colors,
      ylab = "", xlab = "", xaxt = "n", yaxt = "n", bty = "n")

I thought I should do another milkweed, so I chose swamp milkweed - which is lovely dark pink.

iii. The daily family (Asteraceae)

I knew I wanted to make some color palette from something in the Asteraeae (Daisy) family. Sweet coneflower (Rudbeckia subtermosa) is related to brown- and black-eyed Susan.

rudsu_colors <- c("#8D3D46", "#4B2A17", "#F4AB00", "#FEDD00", "#D3BE17", 
                   "#8FA844", "#4C7A3D")
image(1:7, 1, as.matrix(1:7), col = rudsu_colors,
